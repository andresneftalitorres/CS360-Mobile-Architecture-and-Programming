# CS360-Mobile-Architecture-and-Programming

The application I developed was designed to simplify weight tracking, addressing the need for a user-friendly tool to assist users in setting and monitoring their weight objectives.

Key user requirements encompassed effortless data entry, progress visualization, and reminder notifications. To fulfill these requirements, I designed screens for user login, weight entry, settings management, and editing. I prioritized creating an intuitive user interface by employing straightforward forms, prominent buttons, and clear instructions to minimize user confusion.

These designs emphasized user flow, task completion, and visual clarity. During the coding phase, I applied modular design principles by dividing the application into smaller, manageable components and conducted frequent testing to identify and resolve issues. These strategies can be applied in the future to enhance efficiency and debugging ease. I thoroughly tested my code through input validation and ensuring user actions triggered the intended responses.

Testing is crucial for verifying the application’s functionality and enhancing the user experience. I particularly excelled in demonstrating my proficiency in the login and design of a simple, user-friendly application. This showcased my ability to effectively balance functionality and user experience design.
